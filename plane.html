Cessna 395 Skyhawk - RC Plane

Advanced Digital Projects Lab (ECE395) Project

STM32 • Embedded C • IMU Sensor Fusion • RF Systems • PCB Design • Control Systems

A custom-built FPV drone system with wearable IMU-based motion control, integrating embedded flight control, RF video transmission, and custom PCB hardware design.

System Overview

  • Custom STM32-based flight controller
  • 2.4 GHz bidirection radio communcation

My Contributions

  • Designed RF communcation link between transmitter and the plane's flight controller
  • Implemented custom libraries for SX1280 2.4 GHz transciever in embedded C for STM32
  • Designed and fabricated custom PCB integrating RF + sensor systems
  • Built wearable IMU interface for motion-based flight control
